# Flask file distribution

This is a project which tells you. How to distribute a flask project into multipal files.

## project structure
```
.
├── README.md
├── blueprints
│   └── index
│       ├── __init__.py
│       └── views.py
├── extentions
│   ├── server
│   │   ├── __init__.py
│   │   └── server.py
│   └── urls
│       ├── __init__.py
│       └── urls.py
├── main.py
└── requirements.txt
```
